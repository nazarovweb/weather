# Atmospheary - Premium Weather App

Atmospheary is a stunning, modern weather application built with React, Vite, and Framer Motion. It features a premium glassmorphic design system and real-time data fetching from OpenWeatherMap.

## ✨ Features

- **Premium UI**: Dark mode with glassmorphism, vibrant gradients, and rich typography.
- **Dynamic Animations**: Smooth transitions and state changes using Framer Motion.
- **Detailed Metrics**: View temperature, humidity, wind speed, sunrise, and sunset.
- **Smart Search**: Search for any city worldwide.
- **Mock Fallback**: Works immediately with demo data if no API key is provided.

## 🚀 Getting Started

1. **Clone and Install**:
   ```bash
   npm install
   ```

2. **Run Dev Server**:
   ```bash
   npm run dev
   ```

3. **Add API Key**:
   To get real-time weather data, open `src/Weather.jsx` and add your [OpenWeatherMap API Key](https://openweathermap.org/api) to the `API_KEY` variable.

## 🛠️ Tech Stack

- **React 19**
- **Vite**
- **Lucide React** (Icons)
- **Framer Motion** (Animations)
- **Vanilla CSS** (Custom Design System)
